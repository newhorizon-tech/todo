# todo

A simple todo list project

Features

- Display tasks
-  Plain Javascript

## Built With

- HTML/CSS
- Javascript
- Atom

## Getting Started

- Clone the repository
- Navigate to the root directory of the project
- Run 'npm install' to install the dependencies
- Run 'npm run start' to launch the web server
